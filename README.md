# spring-boot-consumer-service

## Tools

- Java version 21
- Maven
- MySQL
- Spring Framework
- Docker

## Dependency

- Spring Boot
- Spring Web
- Spring Security
- Spring Validation
- Lombok
- JDBC API
- Java MySQL Connector

## Installation

Docker:

- clone repo: `$ git clone https://github.com/prafitradimas/spring-boot-consumer-service.git`
- run `docker-compose`:

```sh
docker-compose up -d
```
